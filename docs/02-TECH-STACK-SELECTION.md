## 📋 SUMMARY

**Tech Stack untuk MVP yang SIMPLE, FAST, SCALABLE:**

| Layer | Technology | Why? |
|-------|-----------|------|
| Frontend | React 18 + TypeScript | Popular, mobile-friendly, offline support |
| Mobile | React Native (Expo) | Share code with web, faster dev |
| Backend | Node.js + Express | JavaScript everywhere, fast development |
| Database | PostgreSQL + Supabase | Robust, free, scale easily |
| Auth | Supabase JWT | Built-in, secure, free tier |
| Hosting | Vercel + Railway | Cheap, auto-deploy |
| Offline | Async Storage + SQLite | Offline-first capability |
